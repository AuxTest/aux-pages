FAQ

Часто задаваемые вопросы

Осталось сомнение? Напиши в Telegram — отвечу лично.

Нужно ли уметь программировать?
Нет. Весь курс построен на естественном языке — ты описываешь задачи текстом, AI-ассистент их выполняет. Ни строчки кода писать не нужно. Достаточно опыта в маркетинге стоматологии.
Claude Code или Codex CLI — что выбрать?
Если у тебя macOS/Linux — подойдёт Claude Code (требуется Claude Pro $20/мес). Если Windows — Codex CLI (требуется ChatGPT Plus $20/мес). Оба дают одинаковый опыт. В курсе есть параллельные версии заданий под оба инструмента. Выбирай по своей ОС.
Сколько времени нужно на курс?
13–15 часов чистой практики. Большинство проходит за 1–2 выходных или за неделю по 2 часа вечером. Можно проходить в любом темпе — доступ навсегда.
Что нужно для старта?
1) Подписка на Claude Pro или ChatGPT Plus (от $20/мес). 2) Ноутбук (macOS, Windows или Linux) и интернет. 3) Базовое понимание маркетинга в стоматологии. Node.js устанавливается по инструкции в первом модуле. Никаких специальных знаний не требуется.
А если я из России?
Для Codex CLI (OpenAI) не нужен VPN — он работает на Windows напрямую. Для Claude Code понадобится Claude Pro с зарубежной картой. В курсе есть отдельный онбординг с инструкциями по оплате и установке для студентов из РФ.
Подходит ли курс для сети клиник и для одной стоматологии?
Да. Симулятор построен на примере сети «Зубная Фея» (5 клиник), но все шаблоны и навыки работают и для одной точки. Разница — в объёме данных. В блоке PRO мы отдельно разбираем твой конкретный случай.
